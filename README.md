# 🍫 ChocolateSales
Projeto de análise de dados desenvolvido em Jupyter Notebook utilizando Python, Pandas, Matplotlib e Seaborn. Focado na limpeza, análise exploratória e visualização de dados das vendas, com o objetivo de extrair insights sobre faturamento de cada país, produtos com mais pedidos, e vendedores com maior números de pedidos.

## 🛠️ Tecnologias e Bibliotecas
- **Python:** Linguagem de programação principal para desenvolvimento do projeto.
- **Jupyter Notebooks:** Ambiente para desenvolvimento interativo e documentação do código.
- **Pandas:** Para analisar, manipular e transformar dados.
- **Matplotlib:** Para criação de visualizações estatísticas.
- **Seaborn:** Para gráficos e visualizações personalizadas.

## 🚀 Como Usar

1. **Clone o repositório:**

   ```bash
   https://github.com/RodrygoYamasaki/ChocolateSales.git
   ```

2. **Instale as dependências:**

   ```bash
   pip install -r requirements.txt
   ```

3. **Inicie o Jupyter Notebook:**

   ```bash
   jupyter notebook
   ```

4. **Explore os notebooks:** Para entender o processo e os insights.
